# Booking System Backend

## How to Use

1. **Clone the Repository:**

Download the portable version of MongoDB Community Edition.

Configure MongoDB Path:

Update the config.txt file with the correct path to your mongod.exe executable. For example:

MONGOD_PATH=C:\path\to\mongodb\bin\mongod.exe
Start the Server:

Run the Start Server.bat batch file.

Server Running:

The backend server should now be running.

Test with Postman:

Use Postman to test the API endpoints. You can join the shared Postman workspace using this link: Postman Workspace Invite
